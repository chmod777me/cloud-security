# Cloud Security

This repository documents my practical studies and experiments in cloud security.

The focus is on understanding real-world misconfigurations, identity abuse, privilege escalation paths, and offensive security techniques in cloud environments.

## Objectives

- Understand cloud architecture from a security perspective
- Identify and analyze common misconfigurations
- Study IAM abuse scenarios
- Explore privilege escalation techniques
- Practice offensive cloud methodologies

## Structure

The repository is organized by cloud provider:

- aws/
- azure/
- gcp/

Each folder contains notes, lab summaries, scripts, and practical observations.

## Disclaimer

This repository does not contain solutions or sensitive material from paid platforms.  
All content reflects personal study notes and independent experimentation.
